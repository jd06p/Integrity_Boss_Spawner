# Integrity_Boss_Spawner
A mod that allows Integrity from The Broken Script to spawn in a customized day!


FEATURES:

Custom spawn day: Integrity will spawn in a specific day in a range of 1 and 666 (because 666 its very scary) which you can modify in the config DEFAULT: 66

Ban: If you die because of Integrity, you will be banned from your world (you can disable this in the config) DEFAULT: TRUE
